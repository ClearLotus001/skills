# Excel 边界场景

## 1. 公式与缓存值
- 同时读取公式文本与缓存值。
- 不执行宏。
- 当规则依赖公式结果且缓存值缺失时必须标记问题。

## 2. 合并单元格
- 对合并表头做谨慎处理。
- 规范化表头抽取，避免静默丢列。

## 3. 隐藏 Sheet / 行 / 列
- 在规则或运行配置中显式指定处理策略。
- 保持策略稳定，保证结果可复现。

## 4. 重复表头
- 检测重复列名并作为结构性错误输出。

## 5. 空值语义
- 区分 `null`、空字符串、仅空白字符、业务空值标记。

## 6. CSV 编码
- 解析前探测编码与分隔符。
- 编码无法可靠判定时应中止并提示修复建议。
